Volunteer Nepali Teacher - Summer 2023

In the summer of 2023, I spent two months volunteering at a school in Pyin Oo Lwin, Myanmar. I taught the Nepali language and literature to about 50 excited students in a summer school program.

We taught the students about Nepali stories, poems, grammar and culture. It was amazing to see how eager the students were to learn and how much they improved in just two months. We also gave them tests to check how well they understood the lessons and to help them do better.

This experience was not just about teaching; it also helped me learn about the local culture and community. The students' hard work and the kindness of the people made this volunteering experience very special and unforgettable.

Basic to Advanced OOP with Java

Complete Course 📅 Launched: Oct 2024

What was this course?

This is a complete Java programming course that takes students from knowing nothing about Java to becoming really good at it. I taught everything step by step, making sure students understand each topic before moving to the next one.

Course Level

Starting Level

Complete beginner - no prior programming knowledge needed!

Ending Level

Upper intermediate - ready for small projects!

Course Chapters

1

Basics of OOP with Java

Classes, Objects, Methods, and Constructors, etc.

2

Numerical Data Handling

Performing numerical operations in Java

3

Conditions and Loops

Conditions, Loops and Debugging

4

Arrays and Collections

Arrays, Lists, and Maps along with some interesting algorithms

5

Characters and Strings

Manipulating characters and strings into desired formats

6

Defining Own Classes

Creating custom objects

7

Advanced OOP

Inheritance, Polymorphism, Encapsulation and Abstraction

8

Exception Handling

---

9

File IO

Reading and writing files

10

GUI and Event Driven Programming

Creating graphical user interfaces and handling user interactions

Course Features

Regular quizzes to test understanding and keep students engaged
Hands-on assignments and spontaneous feedback for immediate improvement
Step-by-step teaching from basic concepts to advanced topics
🎉

Course Success!

This was a core Java course that helped many students learn programming from scratch and reach upper intermediate levels! Students loved the interactive quizzes and practical assignments! They got core foundation for Java programming and OOP.
